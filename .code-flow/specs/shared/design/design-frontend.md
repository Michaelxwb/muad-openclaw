### 3.3 组件设计 [必填]

**组件树**（容器/展示分离）

```
<Page>                       # 容器：数据获取 + 状态
└─ <FeaturePanel>            # 容器
   ├─ <ItemList>             # 展示：纯 UI，props 驱动
   │  └─ <ItemRow>           # 展示
   └─ <ActionBar>            # 展示，事件回调上抛
```

| 组件ID | 组件名 | 类型 | 复用来源/去向 | 职责 |
|--------|--------|------|--------------|------|
| CMP-01 | | 容器/展示 | 新建 / 复用现有 | |

> **分层纪律**：容器组件负责数据与状态，展示组件纯 UI（props in / events out）；复用逻辑提取为 hook/composable，禁止跨组件复制粘贴。
